# MotorDC

No arquivo [main](./main.py) contém o espaço de estados
de um motor DC considerando: Corrente de armadura (*ia*),
Velocidade angular (*w_m*) e a Posição (*theta*).

## Parâmetros

1. *Ra*: Resistência de armadura;
2. *La*: Indutância de armadura;
3. *J*: Momento de inercia do motor;
4. *B*: Coeficiente de atrito viscoso;
5. *Ke,Kt*: Coeficiente de força contraeletromotriz e 
mecânica.
